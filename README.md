# 📩 Blacklisted Email Addresses
This repository contains a list of email addresses identified as sources of spam or various types of fraud.
Many of these entries also include the content of the messages.
The purpose of this list is to help administrators identify potential threats and improve email security by proactively blocking or flagging suspicious addresses.

## 📄 How to use this list?
The list of blacklisted email addresses is available in the [LIST.txt](blacklist/LIST.txt) file. Direct raw link:
```text
https://raw.githubusercontent.com/sefinek/Blacklisted-Emails/main/blacklist/LIST.txt
```

## 📨 Where can I preview spam email content?
Markdown files containing spam email content are located in the [blacklist/mails](blacklist/mails) folder.

> [!CAUTION]
> Do not click any links in these messages — they may lead to malicious websites!
